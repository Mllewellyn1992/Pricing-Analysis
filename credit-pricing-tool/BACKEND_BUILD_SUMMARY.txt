================================================================================
API ENDPOINTS
================================================================================

1. POST /api/rate
   Rate a company using both S&P and Moody's engines
   Returns: sp_rating, moodys_rating, blended_rating, metrics

2. POST /api/pricing/lookup
   Look up spreads for a rating + tenor from pricing matrix
   Returns: min_bps, max_bps, mid_bps

3. POST /api/pricing/full-analysis
   Complete analysis: rate company, lookup spreads, compute expected rate
   Returns: ratings, spread, base_rate, expected_range, delta, interpretation

4. GET /api/base-rates
   Get NZ bank base rates (corporate + working capital)
   Returns: List of {bank, corporate_rate, wc_rate, last_updated}

5. POST /api/extract/pdf
   Extract financial data from PDF (stub implementation)
   Returns: status, extracted_fields, filename, message

6. GET /health
   Health check endpoint
   Returns: {status: "healthy"}
