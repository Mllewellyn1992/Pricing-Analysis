================================================================================
ARCHITECTURE OVERVIEW
================================================================================

FastAPI Application Structure:

api/main.py
├── Create FastAPI app
├── Configure CORS middleware
├── Mount routers:
│   ├── api.rate.routes (rating endpoints)
│   ├── api.pricing.routes (pricing endpoints)
│   ├── api.scrape.routes (base rates endpoint)
│   └── api.extract.routes (PDF extraction endpoint)
└── Health check endpoint

Rating Flow (POST /api/rate):

Request (financials + sector_id)
    ↓
Get S&P defaults for sector
    ↓
Run S&P engine (country_risk=2 for NZ)
    ↓
Map sector to Moody's methodology
    ↓
Run Moody's engine (quant_only=True)
    ↓
Blend ratings (average numeric positions)
    ↓
Return: sp_rating, moodys_rating, blended_rating + metrics

Pricing Flow (POST /api/pricing/full-analysis):

Request (financials + sector_id + actual_rate_pct + tenor)
    ↓
Run rating engine (S&P + Moody's)
    ↓
Look up spread from pricing matrix
    ↓
Get NZ base rate
    ↓
Compute expected rate range (base + spread)
    ↓
Calculate delta (actual - expected_mid)
    ↓
Generate interpretation (tight/loose/inline)
    ↓
Return: Full analysis + interpretation
